
## Bella Face Cosméticos:

O projeto foi criado utilizando ReactJS.

Para executar o projeto::

### `yarn install` 

e também 

### `yarn start`

O projeto abrirá no browser.<br />

 [http://localhost:3000](http://localhost:3000)
